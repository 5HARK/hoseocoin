# Hoseocoin App
호서코인 앱
## Requirements
- [Ionic](https://ionicframework.com)
- [Cordova](https://cordova.apache.org/)
- [Typescript](https://www.typescriptlang.org/)

## 설치/빌드/실행 방법
앱 빌드 및 실행 의존성 설치

```
npm install -g ionic cordova
npm install
```

앱 실행
```
ionic serve
```

안드로이드 APK 빌드(개발 버전)
```
ionic cordova build android
```
